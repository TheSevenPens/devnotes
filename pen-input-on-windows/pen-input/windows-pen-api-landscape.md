# Windows Pen API Landscape

A comparison of the available APIs for receiving pen/stylus input on Windows, relevant to drawing and handwriting applications.

<table data-full-width="true"><thead><tr><th>API</th><th>Introduced</th><th>Coordinates</th><th>Precision</th><th>Pen Data</th><th>Cursor Follows Pen</th></tr></thead><tbody><tr><td><strong>Wintab (System)</strong></td><td>1991 (Wacom)</td><td>Physical screen pixels</td><td>Screen resolution (~200 DPI)</td><td>Full: pressure, tilt (azimuth/altitude), twist, buttons, Z (height)</td><td>Yes</td></tr><tr><td><strong>Wintab (Digitizer Hi-Res)</strong></td><td>1991 (Wacom)</td><td>Tablet native units</td><td>Tablet resolution (up to 5080+ LPI)</td><td>Full: pressure, tilt, twist, buttons, Z</td><td>Yes</td></tr><tr><td><strong>WM_POINTER / WM_POINTERUPDATE</strong></td><td>Windows 8</td><td>Physical screen pixels</td><td>Screen resolution¹</td><td>Pressure, tilt (X/Y), twist, buttons via POINTER_PEN_INFO</td><td>Yes</td></tr><tr><td><strong>Windows Ink (WinUI PointerPoint)</strong></td><td>Windows 10 / WinUI</td><td>DIPs (device-independent pixels)</td><td>DIP resolution</td><td>Pressure, tilt (X/Y) via PointerPointProperties</td><td>Yes</td></tr><tr><td><strong>Windows Ink (WPF StylusPoint)</strong></td><td>.NET 3.0 / WPF</td><td>WPF device-independent units</td><td>WPF layout resolution</td><td>Pressure, tilt (X/Y) via StylusPointDescription</td><td>Yes</td></tr><tr><td><strong>RealTimeStylus (COM)</strong></td><td>Windows XP Tablet PC Edition</td><td>HIMETRIC (0.01mm)</td><td>Up to ~2540 DPI²</td><td>Pressure, tilt (X/Y), twist, buttons</td><td>No — raw RTS does not move cursor; frameworks may layer cursor sync on top</td></tr></tbody></table>
